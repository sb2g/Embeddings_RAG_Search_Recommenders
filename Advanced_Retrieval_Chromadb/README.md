## Overview:
- This repo covers some techniques & concepts on Advanced retrieval for AI using Chroma DB, along with my example notebooks from https://learn.deeplearning.ai/courses/advanced-retrieval-for-ai
- It focuses on following topics:
    1. Retrieval
    2. Some issues with it
    3. Query expansion techniques
    4. Cross-encoder re-ranking
    5. Training light-weight embedding adaptors for queries.
